## Default Permission

Default permissions for the plugin

#### This default permission set includes the following:

- `allow-ping`
- `allow-toast`
- `allow-play`
- `allow-pause`
- `allow-reset`
- `allow-next`
- `allow-exit`
- `allow-previewSound`
- `allow-registerListener`
